Lamp 
==============
Lamp V2 is an automation system to control by voice any devices,like lamp, using arduino. 
The second version is dry, small and easy-to-use with [Nodebots.io] (http://nodebots.io/)

To run:
-------

> First:
```bash
 Install [node.js] (http://nodejs.org)
 ```

> Second: 	
```bash
make && make run
```
